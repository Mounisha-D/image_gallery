# SnapGrid Gallery

A smart, modern image gallery with filtering and search capabilities.

## Features

- **40+ Curated Images** - Diverse categories including Nature, Travel, Food, Urban, People, Abstract, and Wildlife
- **Smart Filtering** - Filter by category with visual counters
- **Real-time Search** - Search by title, category, or description
- **Multiple Views** - Grid view and compact list view
- **Lightbox Preview** - Full-screen image viewer with details
- **Dark/Light Mode** - Toggle between themes
- **Responsive Design** - Works on all screen sizes
- **Load More** - Infinite scroll-like functionality
- **Quick Search Tags** - One-click popular searches

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern CSS with Grid, Flexbox, and CSS Variables
- **Vanilla JavaScript** - No frameworks or libraries
- **Font Awesome** - Icon library

## How to Use

1. Open `index.html` in a browser
2. Use search box to find specific images
3. Click category chips to filter
4. Click images to open lightbox view
5. Toggle views between grid and compact
6. Use dark mode for night viewing
7. Click "Load More" to see additional images

## Unique Features

- **Visual Category Counters** - See how many images in each category
- **Image Statistics** - Views and likes for each image
- **Hover Effects** - Smooth animations and overlays
- **Keyboard Navigation** - Use arrow keys in lightbox
- **Smart State Management** - Combined filter and search
